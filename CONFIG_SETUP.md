# 🔧 Настройка конфигурации

## Быстрая настройка

1. **Скопируйте шаблон конфигурации:**
   ```bash
   cp include/config.h.example include/config.h
   ```

2. **Отредактируйте `include/config.h`:**
   - Укажите ваш WiFi SSID и пароль
   - Настройте коррекцию сервоприводов
   - При необходимости измените другие параметры

3. **Соберите и загрузите проект:**
   ```bash
   pio run --target upload
   pio run --target uploadfs
   ```

## Описание параметров

### WiFi настройки
| Параметр | Описание | Пример |
|----------|----------|--------|
| `WIFI_SSID` | Имя WiFi сети | `"MyWiFi"` |
| `WIFI_PASSWORD` | Пароль WiFi | `"secret123"` |

### Сервоприводы
| Параметр | Описание | Пример |
|----------|----------|--------|
| `SERVO_CORRECTION` | Массив коррекции углов (4 значения) | `{-5, -13, -8, -20}` |

### Сеть
| Параметр | Описание | Пример |
|----------|----------|--------|
| `HTTP_PORT` | Порт HTTP сервера | `8080` |
| `AP_SSID` | Имя точки доступа | `"RobotAP"` |
| `AP_PASSWORD` | Пароль точки доступа | `"12345678"` |

## ⚠️ Важно!

- **Файл `include/config.h` НЕ должен быть в git!**
- Он уже добавлен в `.gitignore`
- В репозитории только шаблон `include/config.h.example`
- При клонировании репозитория скопируйте `config.h.example` в `config.h`

## Пример настройки WiFi

```cpp
// Для домашней сети
#define WIFI_SSID "HomeWiFi"
#define WIFI_PASSWORD "MyPassword123"

// Для публичной сети (без пароля)
#define WIFI_SSID "FreeWiFi"
#define WIFI_PASSWORD ""
```

## Пример настройки сервоприводов

Если сервоприводы отклоняются от заданного угла, подберите коррекцию:

```cpp
// Серво 0 отклоняется на +5° → ставим -5
// Серво 1 отклоняется на +13° → ставим -13
// и т.д.
#define SERVO_CORRECTION {-5, -13, -8, -20}
```

Для проверки используйте веб-интерфейс или Serial команды.
