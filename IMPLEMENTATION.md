## 🔌 REST API Reference

### Сервоприводы

#### GET /api/servo
Получить состояние всех сервоприводов.

**Ответ:**
```json
{
  "servos": [
    {"id": 0, "angle": 90, "min": 140, "max": 480},
    {"id": 1, "angle": 90, "min": 140, "max": 480},
    {"id": 2, "angle": 90, "min": 140, "max": 480},
    {"id": 3, "angle": 90, "min": 140, "max": 480}
  ]
}
```

#### POST /api/servo
Установить угол сервопривода.

**Запрос:**
```json
{"id": 0, "angle": 90}
```

### Моторы

#### GET /api/motor
Получить скорость моторов.

**Ответ:**
```json
{"motorA": 0, "motorB": 0}
```

#### POST /api/motor
Установить скорость моторов.

**Запрос:**
```json
{"motorA": 100, "motorB": -150}
```

**Параметры:**
- `motorA`: -255...255 (положительное — вперёд, отрицательное — назад)
- `motorB`: -255...255

#### POST /api/motor/stop
Остановить все моторы.

## 🎨 Веб-интерфейс

### Доступ
```
http://192.168.4.1:8080
```

### Компоненты

1. **Панель сервоприводов**
   - 4 карточки (Servo 0-3)
   - Ползунки для установки угла (0-180°)
   - Кнопки "Применить"
   - Секции калибровки (Min/Max)

2. **Панель DC-моторов**
   - 2 карточки (Motor A, Motor B)
   - Кнопки: ◀ (назад), ■ (стоп), ▶ (вперёд)
   - Отображение текущей скорости
   - Кнопка "Остановить все моторы"

3. **Статус-бар**
   - Индикатор подключения
   - IP адрес робота

## 💻 Примеры использования

### cURL

```bash
# Статус
curl http://192.168.4.1:8080/api/status

# Получить сервоприводы
curl http://192.168.4.1:8080/api/servo

# Установить угол
curl -X POST http://192.168.4.1:8080/api/servo \
  -H "Content-Type: application/json" \
  -d '{"id":0,"angle":90}'

# Моторы
curl -X POST http://192.168.4.1:8080/api/motor \
  -H "Content-Type: application/json" \
  -d '{"motorA":100,"motorB":100}'

# Остановка
curl -X POST http://192.168.4.1:8080/api/motor/stop
```

### Python

```python
import requests

BASE = 'http://192.168.4.1:8080/api'

# Сервопривод
requests.post(f'{BASE}/servo', json={'id': 0, 'angle': 90})

# Моторы
requests.post(f'{BASE}/motor', json={'motorA': 100, 'motorB': 100})

# Остановка
requests.post(f'{BASE}/motor/stop')
```

### JavaScript

```javascript
// Установка угла сервопривода
fetch('/api/servo', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({id: 0, angle: 90})
});

// Управление моторами
fetch('/api/motor', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({motorA: 100, motorB: 100})
});
```

## 📊 Схема подключения

```
ESP32-S3          PCA9685         Motor Driver
────────────────────────────────────────────────
GPIO 8 (SDA)  ─── SDA
GPIO 9 (SCL)  ─── SCL
GPIO 10 (A_IA) ─────────────────── IN1
GPIO 7  (A_IB) ─────────────────── IN2
GPIO 12 (B_IA) ─────────────────── IN3
GPIO 11 (B_IB) ─────────────────── IN4
```

  Сервоприводы:
     - Серво 0 → Мотор B (левая сторона)
     - Серво 1 → Мотор A (правая сторона)
     - Серво 2 → Мотор D (правая сторона)
     - Серво 3 → Мотор C (левая сторона)

    Стороны:
     - Левые: серво 0 (B) и серво 3 (C)
     - Правые: серво 1 (A) и серво 2 (D)

## 🚀 Рекомендации по использованию

1. **Первая загрузка:**
   - Подключите ESP32-S3 по USB
   - Загрузите прошивку через `pio run --target upload`
   - Откройте Serial Monitor (115200 бод)

2. **Подключение к роботу:**
   - Найдите WiFi сеть `netis_2.4G_E86_E86C87`
   - Подключитесь с паролем `X498g8wsd0`
   - Откройте `http://192.168.4.1:8080`

3. **Настройка сервоприводов:**
   - Используйте ползунки для установки угла
   - При необходимости откалибруйте Min/Max
   - Нажмите "Применить"

4. **Управление моторами:**
   - Удерживайте ◀ или ▶ для движения
   - Нажмите ■ для остановки
   - Используйте "Остановить все" для аварийной остановки
